📄 학습 리포트

⚙️ 교사 환경 설정

🔗 Cloudflare Worker 주소를 입력하면 됩니다.
다른 선생님도 Worker URL만 입력하면 바로 사용할 수 있어요!
예) https://xxxxx.workers.dev
🚀 Worker URL 발급 방법 (5분)
① 아래 버튼을 순서대로 클릭하세요:
🌐 2단계: Playground 열기 (새 탭)
💡 Playground가 열리면 기존 코드 전체 선택(Ctrl+A) 후
복사한 코드 붙여넣기(Ctrl+V) → Deploy
③ 오른쪽 위 Deploy 클릭 → 로그인/회원가입
④ Worker 이름 입력 (예: talktalk-english) → Deploy
⑤ 배포 후 Settings → Variables → Add variable
이름: OPENAI_API_KEY
값: platform.openai.com 에서 발급한 Key
→ Encrypt 체크 → Save
⑥ 생성된 Worker URL 위 입력창에 붙여넣기 ✅
💡 다른 선생님도 동일하게!
각자 Cloudflare 가입 → 동일한 Worker 코드 배포 →
본인의 OpenAI Key 입력 → Worker URL 입력하면 끝!
※ OpenAI 비용은 각자 계정에서 청구 (학생 1회 약 7원)
